npm 사용법 
      React 에서 Webpack 실습을 위해서는 CRA(create-react-app) 를 사용해야 한다.

      # CRA 는 create react app 의 약자.

      $ npx create-react-app webpackprj01

      $ cd webpackprj01

      $ ls          # package.json 파일이 있는지 확인

      $ npm run start

      $ cp ../App.css ./src/
      $ cp ../App.js ./src/
      $ cp ../Footer.js ./src/
      $ cp ../Header.js ./src/
